JSP

Jsp标签(<% %>)

 Jsp指示符标签：规定了Jsp页面的整体属性<%@ ... %>

  页面指示符标签<%@ page … %>

    格式:<%@ page [language=“java”] [import=“…”] [errorPage=“path”] [contentType=“mimeType;characterser”] [isErrorPage=“true|false”] [pageEncoding=“characterSer | ISO-8859-1”] %>

  包含指示符标签<%@ include file=“…” %>

   格式：<%@ include file=“path” %>

  标签库指示符标签<%@ taglib … %>

 Jsp脚本标签：在Jsp中嵌入Java代码<% ………….. %>

  实例变量声明<%! 变量声明语句 %>

  表达式<%＝ 表达式 %>

  Java程序脚本<% Java程序代码 %>

 Jsp注释标签

 Jsp动作标签：使用规定的标签将脚本元素中具体的编程细节封住起来。
